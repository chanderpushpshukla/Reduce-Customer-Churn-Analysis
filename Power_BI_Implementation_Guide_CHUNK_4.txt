2.1 TENURE-BASED MEASURES (Q7)

MEASURE 1: Average Tenure by Churn Status
Purpose: Compare average tenure of churned vs. active customers
Audience: Product, Customer Success (identify when churn typically happens)

DAX CODE:
  Avg Tenure Months (All) = AVERAGE('fact_customer_churn'[Tenure_Months])
  
  Avg Tenure Churned = CALCULATE(
    AVERAGE('fact_customer_churn'[Tenure_Months]),
    'fact_customer_churn'[Churn_Flag] = TRUE()
  )
  
  Avg Tenure Active = CALCULATE(
    AVERAGE('fact_customer_churn'[Tenure_Months]),
    'fact_customer_churn'[Churn_Flag] = FALSE()
  )

Comment: These measures reveal if churn is "early-stage" (low avg tenure) or evenly distributed.
Example: If Avg Tenure (Churned) = 4 months, churn is early-stage (focus onboarding).

MEASURE 2: Churn Rate by Tenure Bucket
Purpose: Churn % within each lifecycle stage
Audience: Product, CS Leadership (lifecycle-specific strategies)

DAX CODE:
  Churn Rate % (by Tenure) = [Churn Rate %]
  
  // Comment: [Churn Rate %] measure from Chunk 2 auto-filters by Tenure_Bucket
  // when Tenure_Bucket is in visual context.

For explicit measure (if preferred):

  Churn by Tenure = 
    VAR TenureTotal = DISTINCTCOUNT('fact_customer_churn'[Customer_ID])
    VAR TenureChurned = CALCULATE(
      DISTINCTCOUNT('fact_customer_churn'[Customer_ID]),
      'fact_customer_churn'[Churn_Flag] = TRUE()
    )
    RETURN
      IF(TenureTotal = 0, BLANK(), DIVIDE(TenureChurned, TenureTotal) * 100)

MEASURE 3: Early-Stage Customer Health Score (Derived)
Purpose: Identify risk profile for new customers (0-6 months)
Audience: Onboarding Team (track cohort health)

DAX CODE:
  New Customer Health = 
    VAR NewCustomers = CALCULATE(
      DISTINCTCOUNT('fact_customer_churn'[Customer_ID]),
      'fact_customer_churn'[Tenure_Months] <= 6
    )
    VAR NewCustomersChurned = CALCULATE(
      DISTINCTCOUNT('fact_customer_churn'[Customer_ID]),
      'fact_customer_churn'[Tenure_Months] <= 6,
      'fact_customer_churn'[Churn_Flag] = TRUE()
    )
    VAR EarlyChurnRate = DIVIDE(NewCustomersChurned, NewCustomers)
    RETURN
      IF(EarlyChurnRate > 0.25, "Red", IF(EarlyChurnRate > 0.15, "Yellow", "Green"))

Comment: Categorizes early-stage health (red = >25% churn, yellow = 15â€“25%, green = <15%)
Use Case: Alerts (red = escalate to VP CS)
